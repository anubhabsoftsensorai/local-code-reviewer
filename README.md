# Local Code Reviewer

A powerful, privacy-focused browser extension that performs static analysis and code review entirely in your browser.

## Features
- **Local AST Parsing**: Uses Babel to parse JS/TS and analyze the code structure.
- **Rule-Based Engine**:
  - Cyclomatic complexity estimation.
  - Function length limits.
  - Security checks (`eval`, `innerHTML`).
  - Unused variable detection.
  - Async error handling checks.
  - Modern syntax suggestions (var to let/const).
- **Premium UI**: Side panel with a PR-review style interface, scoring dashboard, and interactive issue cards.
- **Zero Privacy Risks**: All analysis happens locally. No data ever leaves your browser.
- **GitHub Integration**: Automatically injects "Review" buttons next to code blocks on GitHub.

## Tech Stack
- **Frontend**: React + TypeScript + Lucide Icons.
- **Engine**: Babel Parser + Traversal.
- **Build**: Vite + @crxjs/vite-plugin.

## Development

### Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the extension:
   ```bash
   npm run build
   ```
   Or run in dev mode:
   ```bash
   npm run dev
   ```

### Loading into Chrome
1. Go to `chrome://extensions/`.
2. Enable **Developer mode** (top right).
3. Click **Load unpacked**.
4. Select the `dist` folder generated by the build process.

## Architecture
The extension is designed with a shared engine that can be easily ported to other environments (like VS Code).
- `src/engine/`: Core analysis logic.
- `src/ui/`: React-based UI.
- `src/content/`: DOM manipulation and site-specific integration.
- `src/background/`: Extension lifecycle management.
